Обучение основам HTML и CSS

Что представляет из себя HTML и CSS?

HTML - Стандартизированный язык разметки документов для просмотра веб-страниц в браузере. Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора.

CSS (Cascading Style Sheets, каскадные таблицы стилей) — язык описания внешнего вида HTML-документа. Это одна из базовых технологий в современном интернете. Практически ни один сайт не обходится без CSS, поэтому HTML и CSS действуют в единой связке.

Возможности HTML и CSS

Возможности HTML:

  1. Позволяет создавать и структурировать разделы, параграфы, заголовки, ссылки и блоки для веб-страниц.
  2. HTML-документ можно создать в любом редакторе, который есть в операционной системе.
    3. Позволяет создавать таблицы, различные виды списков, добавлять изображения, создавать формы для взаимодействия с пользователем.
  3. Позволяет вставлять аудио и видео контент на страницу.

Возможности CSS:

  1. При помощи таблицы стилей CSS можно изменять поведение HTML элемента: блочный HTML элемент будет вести себя как строчный и наоборот.
    2. Осуществлять выбор элемента или группы элементов, к которым будет применен тот или иной стиль. Осуществлять создание блоков: устанавливать размеры самих блоков, отступов, полей и рамок.
  2. Осуществлять позиционирование блоков.
  3. Управлять шрифтом на странице.
  4. Управлять фоном на странице.

Как создать HTML-документ?

Для того, чтобы создать HTML-документ необходимо:

  • Открыть любой текстовый редактор (Например, блокнот)
  • Разметить HTML основными тегами и набрать произвольный текст (также разметив его тегами)
  • Сохранить файл с расширением .html
